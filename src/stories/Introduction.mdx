# Bem-vindo ao Storybook do Buffs

Este é o Storybook do projeto **Buffs**.

## O que é o Storybook?

O Storybook é uma ferramenta de desenvolvimento que permite:

- **Desenvolver componentes isoladamente** - Trabalhe em componentes UI sem precisar navegar pela aplicação
- **Testar visualmente** - Veja todas as variações de um componente em um só lugar
- **Documentar automaticamente** - Gere documentação a partir do código
- **Compartilhar** - Mostre componentes para designers e stakeholders

## Componentes Disponíveis

Navegue pelas categorias no menu lateral:

### UI Components
Componentes básicos de interface reutilizáveis:
- **Button** - Botões com múltiplas variantes
- **Badge** - Etiquetas de status
- **Table** - Tabelas responsivas
- **Pagination** - Navegação entre páginas
- **MetricCard** - Cards de métricas
- **Loading** - Indicadores de carregamento
- **DashboardContainer** - Containers para dashboards

### Proprietário
Componentes específicos do módulo de proprietário:
- **PropriedadeCard** - Cards para exibir propriedades rurais

## Como Usar

1. **Navegue** pelo menu lateral para encontrar os componentes
2. **Explore** as diferentes variações (stories) de cada componente
3. **Interaja** usando o painel de Controls para modificar props
4. **Teste** em diferentes viewports usando a toolbar
5. **Leia** a documentação automática na aba Docs

## Recursos

### Controls
Modifique as props dos componentes em tempo real para ver como eles se comportam.

### Actions
Veja os eventos disparados pelos componentes (cliques, mudanças, etc).

### Viewport
Teste componentes em diferentes tamanhos de tela.

### Backgrounds
Visualize componentes em diferentes fundos.

## Dicas

- Use o atalho `Ctrl/Cmd + K` para buscar rapidamente
- Alterne entre visualização Canvas e Docs
- Use o addon de Zoom para ver detalhes
- Copie o código das stories para usar na sua aplicação
